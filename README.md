## 以后每节课的整理规则

每次你给一节课 PDF，我会按同一套结构整理：

1. 先生成课程索引，不照搬 PDF。
2. 用中文解释核心概念，保留关键英文术语。
3. 把课程内容拆成可复习的小节。
4. 补充 BA 视角：指标、业务逻辑、分析重点。
5. 最后加复习问题和术语表。

PDF 只作为原始材料，不会逐页复制进仓库。
